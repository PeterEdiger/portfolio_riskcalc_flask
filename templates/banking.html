Schritt für Schritt Anleitung zur Bedienung der Software

  1. Laden Sie sich den Code von GitHub runter
  2. Installieren Sie die requirements.txt Abhängigkeiten
  3. Exportieren Sie eine CSV von ihrem Online-Banking
  4. Definieren Sie die _columns.csv Datei.
    Sollte der Spalten-Name ihrer Bank nicht dabei sein, fügen Sie diesen hinzu.
  5. Definieren Sie Ihre Kategorien in der _categories.csv Datei.
  6. Sie haben drei Skripte zur Verfügung.
    Jedes Skript verfügt über eine eigene Funktion
  7. Beispiel der Ordnerstruktur.
    Die roten Dateien sind die Outputs die Sie erhalten.
  8. Working Directory.
    Definieren Sie für jedes Skript eine Working Directory
    Die Input Dateien werden in die Working directory gelegt.
  9. Beispiel in Pycharm für categorizing_script.py
    Klicken Sie auf Edit configurations von dem Skript
    Stellen Sie hier ihre Working Directory ein.
  10. Beispielbilder Output
  11. Kontakt: go_and_code@web.de
    Sollten sie Fragen oder Anregungen zu dieser
    Software haben kontaktieren sie uns bitte per e-mail.
